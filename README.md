# Úvod
Vítejte ve stručném návodu k vytváření procedurálních realistických povrchů v Blenderu!

V tomto dokumentu se dozvíte jak se tvoří základní materiály tak, aby vypadaly opravdově a realisticky ve vašich dalších modelech. Tento dokument se zabývá tvorbou 4 materiálů a to: ocel, dřevo, asfalt a -VYBRAT-. 

Tento návod byl vytvořen jako praktická část SOČ roku 2025. Veškeré informace o něm lze naleznout v samotné práci.
